# AI_LOG

## Herramientas utilizadas
- ChatGPT (OpenAI)

## Uso de IA
Se utilizó IA para:

- Generar estructuras base de los patrones Strategy y Observer
- Asistir en la implementación inicial de clases
- Ayudar en el diseño de la interfaz gráfica con Swing
- Redactar la documentación técnica

## Modificaciones realizadas
- Adaptación de los ejemplos al dominio del sistema de turnos médicos
- Ajuste de nombres de clases y responsabilidades
- Simplificación de la lógica para cumplir con el alcance del TP1
- Mejora de la interfaz gráfica para simular un sistema real

## Justificación del uso
Se utilizó IA como herramienta de apoyo para acelerar el desarrollo y validar decisiones de diseño.  
El equipo mantuvo control sobre la implementación final, comprendiendo y adaptando cada componente del sistema.

## Conclusión
El uso de IA permitió optimizar tiempos y mejorar la calidad del diseño, manteniendo una comprensión completa del sistema desarrollado.
